# Page Editor Development Server

## Portal config

```
# Avoid checking CSRF
auth.token.check.enabled=false
```

## Steps to run

- `npm install --global page-editor-dev-server`
- Go to page editor directory
- `page-editor-dev-server`

or

- Go to page editor directory
- `npx page-editor-dev-server`

or

- Clone repository
- `npm install`
- `npm link`
- Go to page editor directory
- `page-editor-dev-server`
